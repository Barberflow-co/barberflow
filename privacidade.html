Política de Privacidade

Última atualização: maio de 2025 · Versão 1.0

Resumo simples: Coletamos apenas os dados necessários para o funcionamento do sistema. Nunca vendemos suas informações. Você tem controle total sobre seus dados e pode solicitar exclusão a qualquer momento.

1. Quem somos

O Klipflow é um sistema de gestão desenvolvido e operado pela Klipflow Tecnologia, voltado exclusivamente para barbearias especializadas em prótese capilar no Brasil.

Esta Política de Privacidade descreve como coletamos, usamos, armazenamos e protegemos os dados pessoais de nossos usuários em conformidade com a Lei Geral de Proteção de Dados (LGPD — Lei nº 13.709/2018).

2. Dados que coletamos

Coletamos os seguintes tipos de dados:

  • Dados de cadastro: nome, e-mail, telefone e dados da barbearia
  • Dados de uso: informações sobre como você utiliza o sistema, agendamentos, clientes cadastrados e histórico de atendimentos
  • Dados dos clientes finais: informações inseridas pelo barbeiro sobre seus próprios clientes (nome, telefone, fichas de anamnese)
  • Dados técnicos: endereço IP, tipo de navegador, sistema operacional e logs de acesso
  • Dados financeiros: registros de comandas e faturamento (não armazenamos dados de cartão de crédito)

3. Como usamos seus dados

Utilizamos seus dados exclusivamente para:

  • Fornecer e melhorar as funcionalidades do sistema
  • Processar pagamentos e gerenciar assinaturas
  • Enviar comunicações sobre o serviço (atualizações, alertas importantes)
  • Oferecer suporte técnico
  • Cumprir obrigações legais e regulatórias
  • Garantir a segurança e prevenir fraudes

4. Base legal para o tratamento

O tratamento dos seus dados é realizado com base nas seguintes hipóteses legais previstas na LGPD:

  • Execução de contrato: para fornecer os serviços contratados
  • Legítimo interesse: para melhorar o sistema e garantir a segurança
  • Cumprimento de obrigação legal: quando exigido pela legislação
  • Consentimento: para comunicações de marketing (quando aplicável)

5. Compartilhamento de dados

Não vendemos, alugamos ou compartilhamos seus dados com terceiros para fins comerciais. Podemos compartilhar dados apenas com:

  • Provedores de infraestrutura: Supabase (banco de dados), Vercel (hospedagem) — utilizados exclusivamente para operação do sistema
  • Processadores de pagamento: para processar cobranças de assinatura
  • Autoridades competentes: quando exigido por lei ou ordem judicial

6. Segurança dos dados

Adotamos medidas técnicas e organizacionais para proteger seus dados, incluindo:

  • Criptografia de ponta a ponta em todas as transmissões (HTTPS/TLS)
  • Criptografia dos dados armazenados no banco de dados
  • Controle de acesso baseado em funções (RLS — Row Level Security)
  • Backups diários automáticos com retenção de 30 dias
  • Monitoramento contínuo de acessos e atividades suspeitas

7. Seus direitos (LGPD)

Conforme a LGPD, você tem os seguintes direitos sobre seus dados:

  • Acesso: solicitar confirmação e acesso aos seus dados
  • Correção: corrigir dados incompletos, inexatos ou desatualizados
  • Exclusão: solicitar a exclusão dos seus dados (quando aplicável)
  • Portabilidade: receber seus dados em formato estruturado
  • Revogação: revogar consentimentos anteriores
  • Informação: ser informado sobre com quem compartilhamos seus dados

Para exercer seus direitos, entre em contato pelo e-mail: privacidade@klipflow.com.br

8. Retenção de dados

Mantemos seus dados enquanto sua conta estiver ativa. Após o cancelamento:

  • Dados da conta são mantidos por 90 dias para eventual reativação
  • Após 90 dias, os dados são permanentemente excluídos
  • Dados necessários para cumprimento de obrigações legais podem ser mantidos pelo prazo legalmente exigido

9. Cookies

Utilizamos cookies essenciais para o funcionamento do sistema (autenticação e preferências). Não utilizamos cookies de rastreamento ou publicidade de terceiros.

10. Alterações nesta política

Podemos atualizar esta Política de Privacidade periodicamente. Comunicaremos alterações significativas por e-mail ou através do sistema. O uso continuado após as alterações indica sua aceitação.
